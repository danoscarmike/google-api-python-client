nested(parent, body=None, x__xgafv=None) 
Toggles the nesting of a menu item.

Args:
  parent: string, Required. Name of the menu item. Use the following structure in your
request:
  `organizations/{org}/sites/{site}/menuitems/{menuitem}` (required)
  body: object, The request body.
    The object takes the form of:

{
    "isNested": True or False, # Boolean flag that specifies whether the menu item is a submenu (nested).
  }

  x__xgafv: string, V1 error format.
    Allowed values
      1 - v1 error format
      2 - v2 error format
