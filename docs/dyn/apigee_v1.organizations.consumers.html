access(parent, body=None, x__xgafv=None) 
Checks whether a developer has access to a resource.

Args:
  parent: string, Required. Name of the Apigee organization. Use the following structure in your
 request:
  `organizations/{org}` (required)
  body: object, The request body.
    The object takes the form of:

{
    "data": [ # Resources to check access permissions.
      {
        "action": "A String", # Type of access requested for the provided resource path.
        "resourcePath": "A String", # Resource path.
      },
    ],
  }

  x__xgafv: string, V1 error format.
    Allowed values
      1 - v1 error format
      2 - v2 error format

Returns:
  An object of the form:

    {
    "status": "A String", # Status of the operation.
    "errorCode": "A String", # ID that can be used to find errors in the log files.
    "message": "A String", # Description of the operation.
    "data": [ # Results of the check access requests.
      {
        "action": "A String", # Access that was attempted
        "verdict": "A String", # Results of the check access request.
            # "unknown" is returned if the resource path is not found
        "resourcePath": "A String", # Resource path.
      },
    ],
    "requestId": "A String", # ID that can be used to find request details in the log files.
  }
