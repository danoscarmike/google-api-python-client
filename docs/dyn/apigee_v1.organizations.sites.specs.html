listProxySpecs(name, x__xgafv=None) 
Lists details for the OpenAPI Specifications associated with an API.

Args:
  name: string, Required. Name of the API. Use the following structure in your request:
  `organizations/{org}sites/{site}/specs/{api}` (required)
  x__xgafv: string, V1 error format.
    Allowed values
      1 - v1 error format
      2 - v2 error format

Returns:
  An object of the form:

    {
    "errorCode": "A String", # ID that can be used to find errors in the log files.
    "status": "A String", # Status of the operation.
    "message": "A String", # Description of the operation.
    "data": [ # Details for the OpenAPI Specifications.
      {
        "name": "A String", # Name of the OpenAPI Specification.
        "inDocstore": True or False, # Not used by Apigee hybrid.
        "created": "A String", # Time the portal was created in milliseconds since epoch.
        "url": "A String", # URL of the OpenAPI Specification.
        "lastModified": "A String", # Time the portal was last modified in milliseconds since epoch.
        "proxy": "A String", # Name of the API proxy.
        "revision": 42, # Revision of the API proxy.
      },
    ],
    "requestId": "A String", # ID that can be used to find request details in the log files.
  }
